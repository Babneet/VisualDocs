Steps to Create Pull request:

1. Clone the repository to local repository
   git clone <Repository Http URL from the UI>

git clone https://github.com/Babneet/VisualDocs.git

2. Create a new branch
	git branch <Branch name>

git branch gitdoc

3. Add modify files

	git add . 

4. Commit the changes
	
	git commit -m "Your message here"

5. Switch to master branch

	git checkout master

6. Merger the changes to master branch

	git merger origin master

7. Create a pull request from the UI.
